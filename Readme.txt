지뢰찾기 게임.

CLI 기반 지뢰찾기 게임
가독성을 위해 x, y좌표가 바뀌도록 설정

1. Game Start
실행하면 난이도 선택. 난이도는 Windows MineSweeper 참고.
Windows지뢰찾기처럼 0인 칸은 한번에 주위를 전부 탐색하여 0이 아닌 숫자들 오픈. (DFS로 구현)
깃발 표시 기능. 추측(물음표) 기능은 넣지 않음.
게임클리어 조건은 지뢰가 아닌 땅을 전부 Dig().
클리어하거나 지뢰를 밟으면, Windows지뢰찾기처럼 현재 상황 + 남은 지뢰 위치를 보여준다.

2. Record
기록은 상위 3명까지만 기록. 
매번 저장된 파일을 열고 새로운 상위 기록자가 생기면 기존에 있던 가장 하위 기록을 지우고 채우도록 작성.
